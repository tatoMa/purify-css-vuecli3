# Purifycss

配置在 vue.config.js 里,可以注释全部对比打开和关闭 purifyCss 的区别.对 Bootstrap 而言,可以把 150k 的 css 压缩到 15k 左右,gzip 仅仅 3kb,压缩率很高非常好用.
官方文档参考:https://github.com/webpack-contrib/purifycss-webpack

## Project setup

```
npm install
```
